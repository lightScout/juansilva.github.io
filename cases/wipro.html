// THE BRIEF

A large consumer app needed architectural modernization. Working within a team of five Android developers, I implemented MVVM patterns, replaced MVP presenters with modern ViewModels, and introduced Test-Driven Development to improve code quality and reduce regression risk.

Learned the discipline of working in large teams with shared codebases, code reviews, and architectural consistency.

Java and emerging Kotlin. ConstraintLayout for UI, LiveData and ViewModel for reactive state, Room Database for persistence. JUnit for TDD, custom views and animations for rich UX.

BroadcastReceivers for network monitoring, Glide for image loading, modern Android architecture with proper separation of concerns.
