// THE BRIEF

Puzzle game built entirely in Kotlin Multiplatform. First shipped product under Light Scout — embodying our values of learning through play, human-centred design, and calm, considered digital experiences. Game logic, state, persistence, and IAP backend all shared across platforms. Native UI on both — Compose for Android, SwiftUI integration for iOS.

Full-cycle ownership: design, development, QA, App Store + Play Store submissions, and ongoing maintenance across two independent app store ecosystems.

Kotlin Multiplatform throughout. Shared game engine with Coroutines for frame updates and state management. Koin for dependency injection, Kotlinx Serialization for data models. Multiplatform-settings for local persistence, Firebase Firestore for cloud leaderboards and analytics.

Cross-platform IAP via RevenueCat abstraction — unified monetization layer for StoreKit2 (iOS) and Billing Library (Android). Custom feature flag repository for A/B testing and gradual rollouts.
