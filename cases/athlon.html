// THE BRIEF

Athlon is a vehicle management platform for commercial fleets. I built features across the mobile stack—vehicle tracking, maintenance alerts, driver communications, and compliance reporting.

This was my first professional engineering role. Learned the discipline of shipping code that works, handling edge cases, and supporting users across heterogeneous devices and network conditions.

Java and the Android SDK. Traditional Activities and Fragments, content providers for data sync, SQLite for local storage, and learning to optimize battery and network usage under real-world constraints.

No frameworks yet — just core Android APIs and the hard-won insights about why patterns emerge. Shipped features into production, debugged devices, shipped patches. That foundation shaped everything after.
